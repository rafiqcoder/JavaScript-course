Functions

Functions are blocks of code designed to perform a specific task. They help organize code, avoid repetition, and reduce complexity.

Let's start with a simple example.


// Creating the function

function sum_numbers() {
    var num1 = 5;
    var num2 = 2;
    var sum  = num1 + num2;
    console.log(sum);   
}

// The function is only executed after being invoked:

sum_numbers(); // The console will show 7

Some important observations:

  • Use the function keyword to define a traditional function.
  • Function names follow the same rules as variable names.
  • Parentheses contain parameters (even if empty).
  • Curly braces { } wrap the code block to be executed.
  • Semicolons are not required at the end of function declarations.

Function Arguments

Parameters make functions dynamic. Instead of always summing fixed numbers, we can pass any values: 


function sum_args(num1, num2) {
    var sum = num1 + num2;
    console.log(sum); 
}

sum_args(10, 25);     // 35
sum_args(1000, 375);  // 1375
sum_args(-2, 47);     // 45

The return statement

To reuse the result elsewhere (instead of just printing it), we return the value: 


function sum_args(num1, num2) {
    var sum = num1 + num2;
    return sum; 
}

var average = sum_args(6, 10) / 2;
console.log(sum_args(13, 21));  // 34

ES6+ Function Syntax and Features

Arrow Functions (=>)

Arrow functions offer a shorter syntax and automatically bind this from their surrounding context. 


// Traditional function
function greet(name) {
    return "Hello, " + name;
}

// Arrow function
const greetArrow = name => "Hello, " + name;

console.log(greetArrow("JavaScript")); // Hello, JavaScript

Default Parameters

Set default values for parameters to avoid undefined errors when arguments are missing. 


function multiply(a = 1, b = 1) {
    return a * b;
}

console.log(multiply(4, 5)); // 20
console.log(multiply(7));    // 7

Rest Parameters (...args)

When you're not sure how many arguments a function might receive, rest parameters allow you to capture all of them into an array. 


          function sumAll(...numbers) {
            return numbers.reduce((acc, val) => acc + val, 0);
          }
          
          console.log(sumAll(1, 2, 3, 4)); // 10

Spread Operator

The spread operator lets you pass array elements as individual arguments to a function. 


          const nums = [3, 8, 12];
          console.log(Math.max(...nums)); // 12
        

          const user = { name: "Alice", age: 25 };
const clonedUser = { ...user };

console.log(clonedUser); // { name: "Alice", age: 25 }

Anonymous Functions

Functions can be used without a name, especially in callbacks:


setTimeout(function() {
    console.log("Executed after 1 second");
}, 1000);

// Arrow version
setTimeout(() => console.log("Arrow style"), 1000);
